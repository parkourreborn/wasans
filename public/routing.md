# ROUTING RULES (timetrial run rules)

- Any glitches are prohibited
- Using autoparkour and autotransition is prohibited, automatically invalidates the run
- The clip must show footage of the start of the timetrial (the moment when the click sound is played) and the time the end point as been reached, with all necessary stuff visible in the clip.
  - The necessary stuffs: Properly visible timer, Player's username, Game version shown in bottom of the screen(visible after a death on join)
  - **Very importantly**, the player has to show their fps via "Shift + F5" during the whole run. Otherwise the run will be rejected.

Console users are exempted from this rule as they cannot unlock fps.

- If you're permanently banned from the game, you're ineligible to submit runs for member. This is done to comply with the main game rule.

- Alternative accounts are allowed unless you are banned from the game, as it is considered as ban evasion.

- If the run is specifically world record at the moment, the user is required to submit the run in the acceptable range of time after the run was recorded.
  - For example, reasons such as "was unable to submit cause i had a vacation" or  "gone to bed after the run" may be considered within the acceptable range.
  - Which means, it has to be "Acceptable" on other members' vision; they cannot be justified by reasons such as "wanted to gatekeep it" or "forgot to upload it" <--- (only like if this took a week or dumb long time to submit)
